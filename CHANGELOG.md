# 更新日志

所有重要的项目更改都将记录在此文件中。

## [1.0.0] - 2024-10-21

### ✨ 新增功能

- 🎯 完整的 MCP Server 实现
  - 支持定时健康提醒功能
  - 支持自定义提醒间隔、消息和标题
  - 支持即时发送提醒
  - 支持查询运行状态

- 🖥️ 交互式 MCP Client
  - 友好的菜单式界面
  - 支持默认和自定义配置
  - 实时状态查询
  - 优雅的退出处理

- 🔔 系统通知支持
  - 跨平台系统通知（Windows/macOS/Linux）
  - 可配置提示音
  - 自定义通知内容

- 📚 完善的文档
  - 详细的 README 文档
  - API 参考文档
  - 使用教程
  - 故障排除指南

- 🛠️ 开发工具
  - TypeScript 支持
  - 便捷的启动脚本
  - 示例代码
  - MCP 配置示例

### 📦 依赖项

- @modelcontextprotocol/sdk: ^1.0.4
- node-notifier: ^10.0.1
- typescript: ^5.3.3

### 📝 技术栈

- Node.js
- TypeScript
- Model Context Protocol (MCP)
- node-notifier
